# Resume Builder

A professional and responsive **Resume Builder** web application developed using **HTML, CSS, and JavaScript**.
It enables users to create a clean resume by entering personal details, skills, experience, and education, with support for **profile image upload** and **PDF download**.

### Features

* Dynamic resume generation
* Profile image upload
* PDF export functionality
* Clean and responsive UI

### Tech Stack

* HTML
* CSS
* JavaScript
